# Finsight Finance Analysis Dashboard

## Overview

This Power BI dashboard provides a comprehensive analysis of financial performance, helping stakeholders monitor key financial metrics, identify trends, and make data-driven business decisions.

The dashboard transforms raw financial data into meaningful insights through interactive visualizations, KPI tracking, and trend analysis.

---

## Business Objective

The primary objective of this project is to:

- Monitor overall financial performance
- Analyze revenue and profit trends
- Track key financial KPIs
- Identify high-performing business segments
- Support strategic decision-making through data visualization

---

## Key Performance Indicators (KPIs)

- Total Revenue
- Total Profit
- Profit Margin
- Revenue Growth
- Cost Analysis
